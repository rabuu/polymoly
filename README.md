# PolyMoly
Library and tool for polynomials over abstract rings.

Work in progress.

## Supported rings
- `R`, the real numbers
- `Z`, the integers
- `Z/nZ`, the ring of integers modulo `n` (field if `p` is prime)
- `R[x]`, the polynomial ring over the ring `R`

## Command line interface
```sh
$ polymoly --help
```
